    难点归纳

    • 页面路由,由Event和Components结合,使用监听 .$on or .$emit
    • 跨域,开发环境使用 proxyTable,发布版本结合后台端口,有些API会报304错误
    • 数据优化,数据交流. Vue为数据驱动,直接改变数组不会影响视图的刷新,这里使用的是 push or $set. 数据交流则是使用Event中间与全局变量文件
    • 组件管理. 前期规划中没有使用 router 路由页面,而是使用的eventBus来管理,导致后面管理麻烦.建议使用router来进行管理

    下载使用

    • 下载项目源代码.戳我下载
    • 下载项目服务器依赖.戳我下载
    • 在服务器依赖项目中使用CMD命令(必须在app.js同级目录)set port=8091 & node app.js
    • 在浏览器输入http://localhost:8091/
    • 如果在浏览器中看到运行成功的画面,那么服务器搭建成功
    • 接下来在项目根目录中使用命令安装项目依赖npm install --save
    • 最后在项目根目录中输入命令npm run dev或者打开BAT文件npm run.bat
    • 等待...成功

    项目描述

    使用前端框架Vue开发的一款Web音乐播放器,模仿网易云音乐布局,结合网易云API和酷狗API,有丰富的交互体验.

    顶部

    顶部框架栏,功能模仿网易云音乐

    Logo

    点击刷新页面(需重新登录体验),字体使用时尚中黑简体,Logo是用的网易云音乐的Logo(非商用)

    控制器

    发现音乐界面的前进与后退,用数字来代表当前(0,1,2,3,4)页面. 再用eventBus来发送事件更改发现音乐界面

    搜索

    支持单曲,歌手,歌单,视频,歌词搜索. 单机歌曲播放,歌手进入歌手页面,点击视频进入视频播放页面(有评论,相似mv,mv介绍等),歌词搜索给出的是有关于这个歌词的歌曲

    登录

    点击登录(已登录将显示用户信息,比如粉丝,关注人数...). 说明: 只限于网易云音乐手机登录

    换肤

    点击弹出换肤窗口,有六中颜色可供选择. 会更改侧边栏选择样式,顶部背景,歌单,播放控件等

    左边

    侧边控制栏.有发现音乐,私人FM,MV,朋友,下载管理,我的音乐盘,我的(推荐)电台,我的(创建)歌单

    发现音乐

    • 具有轮播Banner(数据从网易API得到),每日推荐歌单.如果没有登录将显示酷狗最新歌单.
    • 歌单.酷狗API中推荐的最新歌单
    • 排行榜.酷狗API中各种类型的排行榜(飙升榜,TOP500...)
    • 歌手.讲歌手分类,点击进入分类中的歌手,点击歌手进入歌手歌曲界面
    • 最新音乐.酷狗API中的最新音乐

    私人FM

    获取用户私人FM(需登录),点击将播放私人FM,由于一次只拉取三首歌曲,播放到第三首讲再次拉取,送给拉取三次. 有事实歌词和渐变,单击歌手会从酷狗搜索歌手

    MV

    抓去网易MV精选,和MV排行榜. 还有部分MV数据. 单击进入对呀的mv播放界面,有评论,相似mv,mv介绍,mv播放信息

    朋友

    拉取关注列表动态(需登录), 只限于并呈现type为32的动态(网易云官方)

    下载管理

    历史下载记录,和搜索下载.全部都是酷狗音乐,音质为普通

    我的音乐盘

    用户音乐云盘,需要用户上传歌曲到网易云. 单击播放上传的音乐. 上传歌曲请转移到网易云客户端

    我的(推荐)电台

    抓去用户关注的电台和推荐的电台,点击进去可以看到详细的电台列表,双击播放对应的电台

    我的歌单

    拉取用户创建的歌单和关注的歌单,双击播放. 单机全部播放后会清除掉现在播放歌曲的一切信息,并设置为当前歌单的第一个歌曲. 右下角有显示已经添加的歌曲. (相同歌曲不会被添加. 判断条件 url and name)

    歌曲信息

    所有当前播放的歌曲图片,作者,歌曲名. 单机进入歌词界面

    右边

    中间件,页面路由管理器

    底部

    歌曲控制,进度条等功能

    歌曲控制

    上一曲,播放(暂停),下一曲. 换肤后对应的资源也会更换,资源存储在阿里oss(公共读取)

    进度条

    歌曲剩余时间,拖动控件可调节歌曲当前进度. 如果进入了歌词界面,那么拖动歌词也会跳到对应的地点

    音量条

    audio标签的音量,也就是歌曲音量

    播放顺序

    对待播放歌曲进行播放顺序的调整,有单曲循环,列表循环,列表随机

    播放列表

    待播放歌曲,和播放历史记录,双击播放,移入单机垃圾桶可删除单行歌曲
