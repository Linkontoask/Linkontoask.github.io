    网易云(酷狗)音乐

    前提条件

    系统:Windows All

    环境:Node PHP

    下载使用

    • 下载项目源代码.戳我下载
    • 下载项目服务器依赖.戳我下载
    • 在服务器依赖项目中使用CMD命令(必须在app.js同级目录)set port=8091 & node app.js
    • 在浏览器输入http://localhost:8091/
    • 如果在浏览器中看到运行成功的画面,那么服务器搭建成功
    • 接下来在项目根目录中输入命令npm run dev或者打开npm run.bat
    • 等待...成功

    项目描述

    使用前端框架Vue开发的一款Web音乐播放器,模仿网易云音乐布局,结合网易云API和酷狗API,有丰富的交互体验.

    顶部

    顶部框架栏

    Logo

    点击刷新页面(需重新登录体验)

    控制器

    发现音乐界面的前进与后退

    搜索

    支持单曲,歌手,歌单,视频,歌词搜索

    登录

    点击登录(已登录将显示用户信息,比如粉丝,关注人数...)

    换肤

    点击弹出换肤窗口,有六中颜色可供选择

    左边

    侧边控制栏.有发现音乐,私人FM,MV,朋友,下载管理,我的音乐盘,我的(推荐)电台,我的(创建)歌单

    发现音乐

    • 具有轮播Banner(数据从网易API得到),每日推荐歌单.如果没有登录将显示酷狗最新歌单.
    • 歌单.酷狗API中推荐的最新歌单
    • 排行榜.酷狗API中各种类型的排行榜(飙升榜,TOP500...)
    • 歌手.讲歌手分类,点击进入分类中的歌手,点击歌手进入歌手歌曲界面
    • 最新音乐.酷狗API中的最新音乐
